
# GESTION DE VENTE ULTRA

**Développeur** : Tata Camm Solaire  
**Slogan** : _La sécurité sans limites_  
**Mention légale** : PDG DRAMANE TRAORE

Application Flutter hors-ligne pour gérer : produits, ventes, factures, historique, import/export JSON et export PDF.

## Démarrage

```bash
flutter pub get
flutter run
```

## Fonctionnalités
- Tableau de bord (nombre de produits, valeur du stock)
- Gestion des produits (CRUD, image facultative)
- Vente & Facturation (PDF de facture)
- Historique des ventes (export PDF, import/export JSON)
- Graphiques (quantité/valeur)
